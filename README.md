# Project 1: Assignment

This repository contains solutions for a class assignment, consisting of four C programs (`question1.c` through `question4.c`).

## Files

- `question1.c` - `question4.c`: C source files containing the solutions to the exercise questions.
- `students.txt`: Input data file used by the programs in question1.c.
- `sorted_students.txt`: Output file generated by the program1.c.
- `bst_students.txt`: Input data file used by the program in question4.c.

## How to Compile and Run

You can compile any of the questions using a standard C compiler like GCC. For example, to compile and run `question1.c`:

**Windows (PowerShell):**
```powershell
gcc question1.c -o question1.exe
.\question1.exe
```

**Linux / macOS:**
```bash
gcc question1.c -o question1
./question1
```
